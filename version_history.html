Simple Mass Downloader Version History

This page summarizes the most important user-visible changes in recent releases.

Version 0.854

  • Fixed Quick Download Dialog actions after periods of inactivity.
  • Improved handling of pages containing very large inline images.
  • Added clearer notifications when oversized inline images must be omitted.
  • Improved the Batch actions dialog with clearer input hints, selection guidance, and a more usable preview mode.
  • Updated the default file-type categories and added a safe way to restore them from Settings.

Version 0.853

  • Fixed Automatic Folders for downloads started directly from web pages after periods of inactivity.
  • Fixed resource-list filters not being fully restored after periods of inactivity.
  • Added a Version History page accessible from the Help menu.

Version 0.852

  • Improved image detection on modern web pages.
  • Revised interface messages, help pages, and tooltips for clearer and more natural English.
  • Improved popup startup behavior after periods of inactivity.
  • Fixed several interface inconsistencies affecting restored download lists and file status indicators.

Version 0.851

  • Improved popup loading for a smoother and more responsive experience.

Version 0.850

  • Introduced the Manifest V3 version of Simple Mass Downloader, preserving the extension's main download management features on current versions of Chrome.
