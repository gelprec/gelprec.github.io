Simple Mass Downloader Version History

This page summarizes the most important user-visible changes in recent releases.

Version 0.858

  • Fixed Add image to the list for responsive images whose actual displayed URL differs from the image's basic source attribute.
  • Improved Quick Download Dialog readiness handling so its action buttons only become available after the required link data has been received.

Version 0.857

  • Fixed moving selected Download List items with Ctrl+Up and Ctrl+Down.
  • Prevented active download-queue items from being reordered in cases where this could interrupt sequential downloads.
  • Disabled the old Ctrl+S shortcut for starting selected downloads because it conflicted with the browser's Save As command.
  • Added clipboard export actions for checked Resource List items and selected Download List items, making it easier to pass links to an external download manager.
  • Made image selection faster by allowing thumbnails to be selected directly with a click.

Version 0.856

  • Added clearer controls for canceling active download queues.
  • Items waiting in the internal download queue can now be returned to the passive list without being marked as canceled.
  • Clarified that Download List context-menu actions apply only to the item that was clicked.
  • Renamed the open-tabs dialog to make clear that scanned links are added to the current list.

Version 0.855

  • Fixed counter-based name masks so multiple files added to the download list receive sequential numbers.
  • Improved the Batch actions dialog with clearer messages when selected items cannot be changed because downloading has already started.
  • Made completed downloads easier to select by adding a dedicated Open file action.
  • Fixed persisted image size filters so they are applied correctly after loading a new page.
  • Improved Pause all / Resume all behavior for sequential and throttled download queues.

Version 0.854

  • Fixed Quick Download Dialog actions after periods of inactivity.
  • Improved handling of pages containing very large inline images.
  • Added clearer notifications when oversized inline images must be omitted.
  • Improved the Batch actions dialog with clearer input hints, selection guidance, and a more usable preview mode.
  • Updated the default file-type categories and added a safe way to restore them from Settings.

Version 0.853

  • Fixed Automatic Folders for downloads started directly from web pages after periods of inactivity.
  • Fixed resource-list filters not being fully restored after periods of inactivity.
  • Added a Version History page accessible from the Help menu.

Version 0.852

  • Improved image detection on modern web pages.
  • Revised interface messages, help pages, and tooltips for clearer and more natural English.
  • Improved popup startup behavior after periods of inactivity.
  • Fixed several interface inconsistencies affecting restored download lists and file status indicators.

Version 0.851

  • Improved popup loading for a smoother and more responsive experience.

Version 0.850

  • Introduced the Manifest V3 version of Simple Mass Downloader, preserving the extension's main download management features on current versions of Chrome.
